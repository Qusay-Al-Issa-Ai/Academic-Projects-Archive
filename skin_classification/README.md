# Skin Disease Classification System (AI-Powered) 🩺📸

## 📝 وصف المشروع
نظام ذكي يعتمد على الرؤية الحاسوبية (Computer Vision) لتصنيف الأمراض الجلدية من خلال الصور. يقوم النظام بتحليل الصورة المرفوعة وتحديد نوع المرض من بين 3 فئات مختلفة بدقة عالية، مما يساعد في تقديم مؤشرات أولية دقيقة.

## 🛠 التقنيات المستخدمة
* **اللغة:** Python
* **الإطار البرمجي للنموذج:** TensorFlow / Keras.
* **بنية الشبكة:** CNN (Convolutional Neural Network).
* **واجهة المستخدم:** Streamlit (لتقديم تجربة مستخدم تفاعلية وسهلة).
* **معالجة الصور:** OpenCV / Pillow.

## ✨ الأمراض التي يغطيها النظام
تم تدريب النموذج لتصنيف 3أنواع من الحالات الجلدية :
1. حمى (Melanoma).
2. تقرن (Actinic keratosis).
3. سرطان الخلايا القاعدية (Benign_keratosis).


## ⚙️ ميزات النظام
* **تحميل الصور:** واجهة بسيطة لسحب وإفلات الصور.
* **التنبؤ الفوري:** الحصول على النتيجة ونسبة الثقة (Confidence Level) في ثوانٍ.
* **تحليل احترافي:** واجهة مستخدم نظيفة مصممة بـ Streamlit تناسب العروض التقنية.

## 💡 منهجية التطوير (AI-Assisted Development)
بصفتي مهندس تقنية معلومات، قمت بهيكلة الكود البرمجي وتصميم واجهة المستخدم. تم توظيف أدوات الذكاء الاصطناعي للمساعدة في تحسين أداء طبقات الـ CNN ومعالجة البيانات الضخمة، مع الإشراف الكامل على تدريب النموذج وضمان تكامل الـ Backend مع الواجهة البرمجية.
